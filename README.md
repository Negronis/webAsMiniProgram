## Kbone+Vue2.x

### 安装

```shell
npm install
```

***

### 运行

#### Vue端

```shell
npm run serve
```

#### 小程序端

```shell
npm run mp
```

运行指令后，用微信开发者工具打开dist/mp即可运行

***

### 打包

#### Vue端

```shell
npm run build
```

#### 小程序端

```shell
npm run buildmp
```
